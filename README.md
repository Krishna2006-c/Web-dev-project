# 📚 Library Management System

This is a React-based Library Management System that allows users to manage books, members, and notifications efficiently.

## 🚀 Features

* Dashboard with statistics
* Book issue and return system
* Member management
* Notification system
* Login using localStorage
* Responsive UI

## 🛠 Tech Stack

* React.js
* React Router
* CSS
* LocalStorage

## ▶️ Run Locally

```bash
npm install
npm run dev
```
